Project Overview

This project showcases a Node.js web app built and deployed via GitHub Actions and Jenkins.

Technologies Used

  • Node.js
  • Docker & DockerHub
  • Jenkins
  • Git & GitHub

How It Works

  1. Developer pushes code to GitHub
  2. GitHub Actions builds Docker image & pushes to DockerHub
  3. Jenkins pulls image & deploys it
